# aodv-implementation

## Requirements
```
    -   OS      :   Linux_x86_64
    -   Python  :   3.xx
    -   IMDB    :   redis 
```

** This project prefer with Debian OS


## How to run

-   install module in requirements.txt
```
pip3 install requirements.txt 
```

- Running scripts

```
python3 aodv_client.py
python3 aodvAPI.py
```

** Running script is prefer with systemd service
